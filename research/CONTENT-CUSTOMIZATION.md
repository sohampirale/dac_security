# Content Customization Guide

## What to Replace for Your Brand

This document helps you identify all the content that needs to be customized when building the website with your own brand instead of "AKS IT Services".

---

## Company Information

```

NEW: [Your Company Name]

OLD: https://www.aksitservices.co.in/
NEW: https://www.[yourdomain].com/

OLD: info@aksitservices.co.in
NEW: info@[yourdomain].com

OLD: +91-120 454 5911
NEW: [Your Phone Number]

OLD: 1800-120-2394
NEW: [Your Toll-Free Number or remove]
```

```
Create your own privacy policy that includes:
- What data you collect

- Data retention policies

- Contact information
```

```
Create your own terms including:
- Service descriptions
- User obligations
- Limitation of liability
- Dispute resolution
- Governing law
```

```
If you use cookies, create a cookie policy explaining:
- What cookies you use
- Purpose of each cookie
- How users can manage cookies
```

## Color Scheme

### If Changing Colors
```
Current cybersecurity theme:
- Primary: #0a192f (Dark Navy)

2. Update Tailwind config if using Tailwind
```

## Content Tone

### Adjust Based on Your Brand Personality

#### More Formal
```
Use: "Enterprise-grade solutions", "Comprehensive security frameworks"
Avoid: Casual language, contractions
```

#### More Friendly
```
Use: "We're here to help", "Let's secure your business together"
Avoid: Overly technical jargon, complex sentences
```

#### More Technical
```
Use: Specific technologies, standards, methodologies
Avoid: Marketing fluff, vague promises
```

## Analytics & Tracking

### Set Up
```
1. Google Analytics 4
   - Create GA4 property
   - Add tracking code to all pages
   - Set up conversion events

2. Google Search Console
   - Verify domain ownership
   - Submit sitemap
   - Monitor search performance

3. Google My Business
   - Create/optimize listing
   - Add photos
   - Collect reviews

4. Heat Mapping (Optional)
   - Hotjar or Microsoft Clarity
   - Track user behavior
   - Identify improvement areas
```
