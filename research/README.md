# AKS IT Services Website Research Documentation

## Overview
This directory contains comprehensive research of the AKS IT Services website (https://www.aksitservices.co.in/) for creating a similar static website with Next.js.

## Project Scope
Create a 3-4 page static website with:
1. **Home Page** - Main landing page
2. **Services Page** - All service offerings
3. **Products Page** - All product offerings
4. **Contact Us Page** - Contact information

## Research Contents

### 1. `homepage-content.md`
Complete content extraction from the homepage including:
- Navigation structure
- Hero section
- About/Commitment section
- Services overview
- Statistics
- Trust and certifications
- Client testimonials
- Call-to-action sections
- Footer content

### 2. `services-content.md`
Detailed breakdown of all 7 service categories:
- Application Security Assessment
- Network Security Assessment
- Compliance and Data Privacy
- Red Teaming
- IoT Security Assessment
- Security Operation Center (SOC)
- Digital Forensic Services

### 3. `products-content.md`
Complete product catalog including:
- IT Security Products
- Partner Companies
- Digital Forensic Products
- Forensic Software Tools

### 4. `contact-information.md`
All contact details and company information

### 5. `site-structure.md`
Information architecture and navigation structure

### 6. `design-elements.md`
Visual design elements, color schemes, and layout patterns

### 7. `seo-keywords.md`
Keywords and meta information for SEO
